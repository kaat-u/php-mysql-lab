# PHP MySQL LAB

Repositorio con los laboratorios prácticos del curso, organizados por número y tema. Cada carpeta contiene un ejercicio independiente enfocado en PHP y MySQL.

## Tecnologías

- PHP
- MySQL
- XAMPP

## Estructura del proyecto

```
LAB01-php/
LAB02-elecciones-db/
LAB03-api/
LAB04-json/
LAB05-tienda-db/
LAB06-filtro-categorias/

LAB08-veterinaria/

LAB11-academia-landing-page/
LAB12-carrito-compras/
empresa-db/

```

## Laboratorios

| Carpeta | Tema |
|---|---|
| `LAB01-php` | Fundamentos de PHP |
| `LAB02-elecciones-db` | Conexión a base de datos: sistema de elecciones |
| `LAB03-api` | Consumo/construcción de una API |
| `LAB04-json` | Manejo de datos en formato JSON |
| `LAB05-tienda-db` | Base de datos de una tienda |
| `LAB06-filtro-categorias` | Filtrado de productos por categoría |
| `LAB07-buscador-productos` | Buscador de productos |
| `LAB08-veterinaria` | Sistema para una veterinaria |
| `LAB09-login` | Sistema de inicio de sesión |
| `LAB10-crud-pdo` | CRUD usando PDO |
| `LAB11-academia-landing-page` | Landing page para una academia |
| `LAB12-carrito-compras` | Carrito de compras |
